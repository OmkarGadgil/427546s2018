Computer Graphics Assignment 1.

Open the file MainMenu.html for accessing the assignment.

The program has been tested in Mozilla Firefox and Google Chrome Browser.

the assignment allows us to draw a line, circle, ellipse, rectangle, polygons and polylines(with rubberbanding).
